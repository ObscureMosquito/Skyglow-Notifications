Skyglow Notifications Guide

  • Step 1: Enable Notifications

    In order to receive notifications, you must click on the enable remote notifications button, and select which app you want to do it for.

  • Step 2: Configure Server

    Enter your notification server's address and port. These are crucial for receiving notifications.

  • Step 3: Generate SSL Certificates

    Click on the generate SSL certificates button and start the certificate secure transfer process, on the server, start the server with the "setup" argument.

  • Step 3: Apply and Respring

    After configuring your settings, tap 'Apply and Respring' to save your changes and respring your device.

Server Setup

In order to set up the server, you must download the source code from the GitHub page, deploy it in a Python3 compatible system, open your desired network port, and follow the instructions below.

Warning: These instructions are better followed and read on the GitHub readme.

  • Step 1: Download & Setup

    Download the Python Notification server, and set up your application/server to send requests to the notification server in the expected payload whenever you want a notification to be sent to the user.

  • Step 2: Netowrk Configuration

    Open the necessary network ports or allow the traffic on your firewall, this will depend on your setup. (Not reccomended to use the default TCP port)

  • Step 3: Automation Setup

    Set up your server to run on start up automatically (if desired), most importantly, allow it to restart every 6 hours or so, this is not 100% required but will greatly improve stability On Debian systems this can be done with systemd.
